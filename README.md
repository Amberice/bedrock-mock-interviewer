# Stellar Interviewer 🚀
An AI-powered mock interview application built with React, Amazon Bedrock (Nova Micro), and DynamoDB.

## 📅 Project Roadmap (7-Week Full-Stack Sprint)

### Week 1: Core Brain & Infrastructure
- Set up AWS CDK for infrastructure as code.
- Deployed AWS Lambda backend.
- Integrated Amazon Bedrock (Nova Micro model).

### Week 2: Persistent Memory
- Set up Amazon DynamoDB for chat history.
- Implemented Session ID logic to separate user conversations.
- Configured IAM roles for secure access.

### Week 3: Modern UI/UX
- Scaffolded frontend with React + Vite + TypeScript.
- Styled with Tailwind CSS for a professional look.
- Implemented auto-scrolling and loading states.

### Week 4: Voice & Speech Integration
- Integrated Web Speech API for Text-to-Speech (Listen).
- Implemented Speech-to-Text (Microphone input).
- Added markdown cleaning for cleaner voice readouts.

### Week 5: Cloud Deployment
- Deployed the frontend to AWS Amplify.
- Wired production API endpoints and environment configuration.
- Stabilized deployment flow and fixed deployment-related issues.
- Improved app readiness for public hosting and HTTPS access.

### Week 6: PWA & Mobile Experience
- Enabled Progressive Web App support with `vite-plugin-pwa`.
- Added manifest, service worker registration, and installable app behavior.
- Added custom app icons, favicon assets, and mobile home screen support.
- Updated HTML metadata and icon references for browser/mobile compatibility.

### Week 6.1: Prompt & UX Stabilization
- Refined interview prompts and response behavior.
- Fixed frontend/backend issues affecting chat flow and app stability.
- Updated UI logic and infrastructure configuration tied to the latest release.

### Week 7: Final Polish & Portfolio Prep (Upcoming)
- Add observability, usage monitoring, and error tracking.
- Complete end-to-end testing for interview flow and edge cases.
- Finalize documentation, architecture notes, and portfolio presentation.
